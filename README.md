# Calculator
This is a simple calculator web application built using HTML, CSS, and JavaScript.

Features
The calculator can perform basic arithmetic operations such as addition, subtraction, multiplication, and division. 
It also includes the ability to perform decimal point calculations and clear the input field.

Usage
To use the calculator, simply click the number buttons to input numbers, click the operation buttons to select an operation, 
and click the equals button to display the result. You can also click the "C" button to clear the input field or 
the "DEL" button to delete the last entered digit.

Installation
To install the calculator, simply download or clone the project files and open the "index.html" file in your web browser.
